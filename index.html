Mercury

Einfach zusammenarbeiten.

Mit konfigurierbaren Gruppenchats, einem automatischen Kalender, Cloudspeicher und einer globalen Suchfunktion

Gruppenchats bieten eine einfache und schnelle Möglichkeit, mit Ihrer gesamten Gruppe zu kommunizieren. Egal, ob Sie mit Ihrer Familie einen Ausflug planen, Ihren Kollegen etwas mitteilen möchten oder Schüler unterrichten. Senden Sie Fotos, Videos und andere Dateien oder heben Sie die Stimmung mit einem Sticker.

Als Administrator einer Gruppe können Sie festlegen, was Mitglieder senden dürfen und sogar Ruhezeiten bestimmen, während denen nichts geschrieben werden kann.

Der integrierte, automatisch erstellte Kalender sorgt dafür, dass Sie nie wieder Aufgaben manuell eintragen und sie dann selbst als erledigt kennzeichnen müssen. Mercury erledigt all das für Sie.

Erstellt jemand eine Aufgabe mit Frist für Ihre Gruppe, wird sie automatisch in den Kalender eingetragen. Das gleiche geschieht bei Bekanntmachungen.

Dateien freigeben und verwalten: Wenn Sie etwas mit anderen teilen möchten, laden Sie es einfach hoch und schon ist es für alle verfügbar. Mit dem umfassenden Rechtesystem können Sie genau festlegen, wer Zugriff auf Ihre Dateien erhält.

Mit den enthaltenen Dateiwerkzeugen können Sie ganz einfach aufräumen und Ihren Speicher strukturieren.

Aufgaben verteilen und verwalten: Für Bildungseinrichtungen bietet Mercury ein einfach zu bedienendes und dennoch vielseitiges Aufgabensystem. Geben Sie direkt Rückmeldungen, erhalten Sie Benachrichtigung über abgegebene Lösungen und behalten Sie den Überblick mit der Integration in den Kalender.

Auch für private Haushalte können Aufgaben nützlich sein: Was muss wann getan werden?

Finden Sie alles, wann immer Sie wollen. Die globale Suche, mit der Sie Dateien, Gruppen, Nachrichten, Kalendereinträge und Aufgaben ganz einfach finden können, ist immer nur einen Klick oder einen Shortcut entfernt.

Möchten Sie genau steuern, was Sie suchen? Mit der erweiterten Suche legen Sie Filter fest und kommen so präzise zum Ziel.

Steuern Sie die Medienwiedergabe, überprüfen Sie anstehende Aufgaben und antworten Sie sofort auf Nachrichten - mit Widgets können Sie von jedem Bildschirm aus sofort das Heft in die Hand nehmen.

Widgets erscheinen immer dann automatisch, wenn Sie sie benötigen: Wenn eine Mediendatei abgespielt wird, erscheint ein Medienwidget, wenn Sie in einer Nachricht erwähnt werden, sehen Sie ein Widget, um schnell zu antworten. So können Sie sich voll und ganz auf das konzentrieren, was sie gerade tun.

Überall nutzen.

Als webbasierte Software läuft Mercury auf jedem Gerät, das einen modernen Browser unterstützt - nutzen Sie Mercury mit Windows, macOS, Linux, Android, iOS (und überall, wo man Firefox, Chrome oder Safari nutzen kann).

Mercury wurde von Grund auf als Progressive Web App (PWA) entwickelt und lässt sich daher auf Ihren Geräten wie eine normale App installieren und offline nutzen. Ziehen Sie es vor, Mercury im Browser zu nutzen, steht Ihnen diese Option natürlich auch offen.
